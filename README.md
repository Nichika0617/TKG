卒業要件を満たすために必要な単位が確認できるサイトです．

成績表に書かれた値を入力すると
卒業要件との差分，選択するべき単位数が確認できます．

## エレベーターピッチ
```
履修登録がよく分からない，卒業できるかの不安を減らしたい

卒業要件をしっかり把握したい人 向けの

履修登録の不安を減らすこと を実現する

 TKG です。
 
これは  本当に選択するべき科目の把握，個数の確認ができ

既存の代替品 はなく，

顧客のニーズをすでに把握しているという圧倒的な優位性  が備わっています。
```
